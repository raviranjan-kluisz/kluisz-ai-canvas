There are several ways you can interact with the Kluisz community and learn more about the Kluisz codebase.

## Join the Kluisz Discord server

Join the [Kluisz Discord Server](https://discord.gg/kluisz) to ask questions and showcase your projects.

## Follow Kluisz on X

Follow [@kluisz_ai](https://x.com/kluisz_ai) on X to get the latest news about Kluisz.

## Star Kluisz on GitHub

If you like Kluisz, you can star the [Kluisz GitHub repository](https://github.com/kluisz/kluisz-ai-canvas).
Stars help other users find Kluisz more easily, and quickly understand that other users have found it useful.

Because Kluisz is open-source, the more visible the repository is, the more likely the codebase is to attract [contributors](/contributing-how-to-contribute).

## Watch the GitHub repository

You can watch the [Kluisz GitHub repository](https://github.com/kluisz/kluisz-ai-canvas) to get notified about new releases and other repository activity.

To get release notifications only, select **Releases only**.

If you select **Watching**, you will receive notifications about new releases as well as issues, discussions, and pull requests, if you are interested in that activity.
For information about customizing repository notifications, see the [GitHub documentation on repository subscriptions](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions).

## Request enhancements and get help through GitHub

You can also submit feature requests and get help with Kluisz through the GitHub repository.
For more information, see [Get help and request enhancements](/contributing-github-issues).
